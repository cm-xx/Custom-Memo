# 自定义备忘录 Chrome 插件

一个简单而强大的 Chrome 新标签页备忘录插件，帮助您随时记录和管理重要信息。

## 功能特点

- 🎯 将新标签页转换为个性化备忘录
- 📝 支持实时编辑和保存
- 💾 使用 Chrome 存储 API 自动保存数据
- 🎨 美观的用户界面
- 🔒 数据本地存储，保护隐私

## 安装方法

1. 从 Chrome 网上应用店安装（即将上线）
2. 手动安装：
   - 下载此仓库的 ZIP 文件并解压
   - 打开 Chrome 浏览器，进入扩展程序页面 (chrome://extensions/)
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹

## 使用说明

1. 安装插件后，打开新标签页
2. 直接在页面上编辑您的备忘录内容
3. 内容会自动保存
4. 您可以随时查看和编辑您的备忘录

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chrome Extension API
